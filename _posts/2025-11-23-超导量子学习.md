# 定义哈密顿量

广义动能：磁通量
广义势能：电场能量
![image.png](https://cnblog-img-langle.oss-cn-beijing.aliyuncs.com/obsidian-img/2025/11/23/fbbd7aedd376853a8b8ab9ab9df27bbf.png){: width="250" }

LC谐振电路：能量在电容 $c_{r}$ 的电能和电感 $L_{r}$ 的磁能中转换
因此定义
$\qquad$ 1. 广义动能：电场势能
$\qquad$ 2. 广义势能：磁能
遵循Hamiltonian描述
LC回路能量公式：
$$E ( t ) = \int _ { - \infty } ^ { t } V ( t ^ { \prime } ) I ( t ^ { \prime } ) d t ^ { \prime } \ \tag{1-1} 

$$
将电压随时间变化定义为一个flux：
$$\Phi(t) = \int_{-\infty}^t V(t') dt' \tag{1-2}$$
因为在LC谐振电路中:
$$V = L \frac{dI}{dt} \tag{1-3}$$
$$I = C \frac{dV}{dt} \tag{1-4}$$
将 $\text{(1-1)}$ 、$\text{(1-2)}$ 带入 $\text{(1-3)}$ 、$\text{(1-4)}$ 得, 用flux表示动能、势能
$$\mathcal{T}_{C} = \frac{1}{2} C \dot{\Phi}^{2}, \tag{1-5}$$
$$\mathcal{U}_{L} = \frac{1}{2L} \Phi^{2}. \tag{1-6}$$

> [!NOTE]- 推导
> $$ 
>  \begin{align}
>  \mathcal{T}_{C} &= \int {cV \frac{dV}{dt}}  \\
>                  &= \frac{1}{2} C \dot{\Phi}^{2} \\
> \mathcal{U}_{L} &= \int \frac{V^2}{L} dt   \\
>                &= \frac{1}{2L} \Phi^{2}
>  \end{align}  
> {{}}  $$

$$
\begin{align}
\mathcal{L}(Lagrangian) &= 动能 - 势能 \\
    \mathcal{H}(Hamiltonian) &= 动能 + 势能
\end{align}
{{}}$$
因此
$$\mathcal{L}=\mathcal{T}_C-\mathcal{U}_L=\frac{1}{2}C\dot{\Phi}^2-\frac{1}{2L}\Phi^2. \tag{1-7}$$
第二项是 $\dot{\Phi}$ 对时间的定积分
$$Q=\frac{\partial\mathcal{L}}{\partial\dot{\Phi}}=C\dot{\Phi}. \tag{1-8}$$
$$H=Q\dot{\Phi}-\mathcal{L}=\frac{Q^2}{2C}+\frac{\Phi^2}{2L}\equiv\frac{1}{2}CV^2+\frac{1}{2}LI^2, \tag{1-9}$$
为了满足量子力学描述
$\qquad$ $Q\to P$ 
$\qquad$ $\Phi \to x$ 
$\qquad$ $C \to m$
$\qquad$ $w = \sqrt{ LC }$
因此 $H$ 类似谐振子模型（见格里菲斯第二章谐振子模型）
$$H=\frac{1}{2m}\left[p^2+(m\omega x)^2\right]. \tag{1-10}$$
对易关系 $$[\hat{\Phi},\hat{Q}]=\hat{\Phi}\hat{Q}-\hat{Q}\hat{\Phi}=i\hbar \tag{1-11}$$

> [!NOTE]- 对易关系推导 $[\hat{\Phi},\hat{Q}] = i\hbar$
